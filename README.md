## Inicio rapido

### Opci√≥n A ‚Äî Docker (recomendado)

Requiere √∫nicamente tener [Docker](https://docs.docker.com/get-docker/) instalado.

```bash
docker run -d \
  --name tupisec \
  -p 3000:3000 \
  -v tupisec-data:/app/dashboard/data \
  --restart unless-stopped \
  ghcr.io/idod00/tupisec:latest
```

Abre [http://localhost:3000](http://localhost:3000) ‚Äî listo.

Para actualizar a la √∫ltima versi√≥n:

```bash
docker pull ghcr.io/idod00/tupisec:latest
docker rm -f tupisec
docker run -d \
  --name tupisec \
  -p 3000:3000 \
  -v tupisec-data:/app/dashboard/data \
  --restart unless-stopped \
  ghcr.io/idod00/tupisec:latest
```

> Los datos (base de datos y screenshots) se persisten en el volumen `tupisec-data` y **no se pierden** al actualizar.

- `pip` y `npm`

#### 1 ‚Äî Clonar el repositorio

```bash

cd TUPISEC
```

#### 2 ‚Äî Configurar el scanner (Python)

```bash

source venv/bin/activate          # Windows: venv\Scripts\activate

```

```bash

npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Funcionalidades del dashboard

- **Nuevo scan** ‚Äî lanza el scanner desde el browser con progreso en tiempo real
- **Historial** ‚Äî todos los scans agrupados por dominio con sparkline de tendencia de score
- **Reporte de seguridad** ‚Äî findings organizados por severidad, score, graficos coloreados por severidad
- **OWASP Top 10** ‚Äî cobertura mapeada a las 10 categorias OWASP 2021 en cada reporte
- **Comparacion** ‚Äî compara dos scans del mismo objetivo para ver progreso
- **Batch scan** ‚Äî escanea multiples URLs en paralelo
- **Export** ‚Äî descarga reportes en PDF, CSV o JSON
- **Notas en findings** ‚Äî agrega estado de remediacion y notas por hallazgo
- **Tech stack + CVEs** ‚Äî detecta tecnologias y consulta CVEs criticos en NVD
- **Subdominios** ‚Äî tabla de subdominios encontrados con riesgo de takeover
- **Fuzzing results** ‚Äî tabla de parametros ocultos que cambiaron el comportamiento
- **Threat Intelligence** ‚Äî enriquecimiento con VirusTotal y Shodan (API keys opcionales)
- **SSL Monitor** ‚Äî monitoreo programado de certificados con alertas por webhook y email
- **Notificaciones** ‚Äî webhooks y Slack al completar un scan o detectar hallazgos criticos
- **Escaneos programados** ‚Äî re-escaneo automatico diario, semanal o mensual
- **Modo claro / oscuro** ‚Äî toggle en la barra de navegacion, preferencia persistida en el navegador
